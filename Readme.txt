Adds academic structure to Tutor LMS with visibility control.

== Description ==
This plugin adds hierarchical academic classifications to Tutor LMS, including universities, schools, and general courses. It restricts course visibility based on user classification.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to Tutor LMS settings to configure.
